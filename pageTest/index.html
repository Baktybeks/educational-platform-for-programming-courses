Изучи Java Script

JavaScript — мультипарадигменный язык программирования. Поддерживает объектно-ориентированный, императивный и функциональный стили. Является реализацией спецификации ECMAScript. JavaScript обычно используется как встраиваемый язык для программного доступа к объектам приложений.

Начало

Скачиваете и заходите Visual Studio Code настраеваите окружение под вашу удобную работу
создаете фаилы index.html,style.css и script.js

  • Подлючаете все файлы stye.css и script.js к index.html и открываем его
  • Открываете script.js на первой строчке пишете
  • Следующую команду console.log("Hello World"); и открывает консоль в браузере
  • Ура!! Поздровляем свашеей первой программой
